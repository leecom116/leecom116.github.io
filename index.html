- Ajax를 적극 활용하여 모든 기능을 하나의 jsp안에 구현

- Oracle을 이용한 트랜잭션 처리

- Python과 MongoDB를 이용하여 데이터 가공

- GoormIDE Container에 Flask를 활용한 자체 서버 구현

- json 형식으로 카카오 챗봇 서버와 자체 챗봇 서버간의 데이터 공유

WBS 일정관리 프로그램

WBS(Work Breakdown Structure)는 프로젝트를 효율적으로 진행하기 위해 업무 일정을 계획하고 관리할 수 있는 일정관리 프로그램입니다.

프로젝트의 작업단위를 프로젝트, 대분류, 중분류, 소분류 총 4단계의 트리구조로 세분화하여 업무를 쉽게 소화할 수 있도록 구현하였습니다.

- 트랜잭션 처리를 통한 데이터 무결성

- 예외처리를 이용하여 비정상 종료 방지

- 정규식을 사용하여 데이터 정확성 향상

- JDBC Driver를 이용하여 Java와 Oracle을 연결

개발환경

- Java8, Oracle 18C, Eclipse, SQL Developer, GitHub
